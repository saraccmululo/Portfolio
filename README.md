# Personal Portfolio Web App

Responsive portfolio website showcasing projects, skills, and contact information. Designed to highlight web development work with interactive features and smooth navigation.

---

## Features
- Home, About, Projects, and Contact sections  
- Interactive project showcase with filters and reveal-on-scroll animations  
- Responsive navigation with mobile-friendly menu  
- Loading screen for smooth user experience  
- Fully responsive design for desktop and mobile  

---

## Technologies
React.js, Vite, Tailwind CSS, CSS3

---

## Installation
1. Clone the repository:
   git clone https://github.com/saraccmululo/portfolio.git
   cd portfolio
2. Install dependencies:
    npm install
3. Start the development server:
    npm run dev
    Open your browser at the URL shown in the terminal 

Deployment
Live website: https://saraccmululo.github.io/portfolio/#projects

Usage
Explore the Home and About sections to learn about skills and experience
View Projects section for detailed project showcases
Use Contact section to reach out via email or social links
Navigation adapts seamlessly for desktop and mobile devices
